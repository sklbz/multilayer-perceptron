## Installation

Ensure you have Rust installed. If not, install it using [rustup](https://rustup.rs/):

```sh
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
Then, add this crate to your project:
```sh
cargo add ...
```

Or manually add it to your Cargo.toml
```toml
[dependencies]
...
```
